# Opis projektu
System monitoruje wykonaywanie martwego ciągu w czasie rzeczywistym,
wykorzystując dwie niezależne kamery,
co pozwala na szybką ocenę poprawności wykonywania ćwiczenia.
Aplikacja wykrywa odchylenia od wzorca i informuje użytkownika o błędzie.
